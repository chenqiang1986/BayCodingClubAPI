# BayCodingClub API Repository

## Setup instruction

### Github Desktop
https://desktop.github.com/download/

Clone this repository to your local computer using Github Desktop

### IDE Setup (Optional)
https://code.visualstudio.com/


### Python Setup
Make sure running the command in Terminal gives you the python version.
```
python3 --version
```

If you observe something like "unknown command python3", please first install python on your computer.

https://www.python.org/downloads/

### Dependency

```
pip3 install -r requirements.txt
```
