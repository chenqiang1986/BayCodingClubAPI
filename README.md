# BayCodingClub API Repository

## Setup instruction

### Github Desktop
https://desktop.github.com/download/

1. Create your own github repository. (Not clone this one!!!)
3. Copy exact content of .gitignore and requirements.txt from this repository.
3. Create file .gitignore and requirements.txt in your repository with same content.
4. Push your change to your remote github repository. 

### IDE Setup (Optional)
https://code.visualstudio.com/


### Python Setup
Make sure running the command in Terminal gives you the python version.
```
python3 --version
```

If you observe something like "unknown command python3", please first install python on your computer.

https://www.python.org/downloads/

### Dependency

```
pip3 install -r requirements.txt
```
